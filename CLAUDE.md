# CLAUDE.md

## Project Overview
A Streamlit-based 2D structural bar calculation app for engineering education.
Student: Pop Rareș Darius

## Tech Stack
- Python + Streamlit (UI)
- NumPy (calculations)
- Matplotlib (plots, PDF export)

## Key File
- `app.py` — entire application in a single file

## Running the App
```bash
streamlit run app.py
```

## Notes
- UI and comments are in Romanian
- The app computes 2D bar geometry, cross-section properties, and structural analysis
