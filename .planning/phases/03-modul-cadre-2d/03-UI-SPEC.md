## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps in toolbar, inline label padding |
| sm | 8px | Toolbar button gaps, side panel field spacing |
| md | 16px | Default element spacing, canvas-to-panel gap |
| lg | 24px | Side panel section padding, tab bar padding |
| xl | 32px | Page-level horizontal padding |

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 500 (medium) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 600 (semibold) | 1.2 |

**Notes:**
- Body 14px used for side panel field labels, tooltips, result table cells.
- Label 12px used for SVG diagram value labels (max/min per bar), grid axis labels, toolbar mode indicator.
- Heading 20px used for page title "Cadru 2D" and side panel section headings ("Nod", "Bara").
- Display 28px not used in this phase (no hero content). Declared for consistency with Phase 1 design system.
- Font: Geist Variable via `@fontsource-variable/geist` (already imported in `index.css`).
- SVG text elements use `font-family: 'Geist Variable', sans-serif` and `font-size` in px matching the scale above.

## Interaction Contract

### Editor State Machine

| Current Mode | Event | Action | Next Mode |
|-------------|-------|--------|-----------|
| add_node | Click empty canvas | Add node at snapped coordinates | add_node |
| add_node | Click existing node | Select node, open side panel | select |
| add_bar | Click node (1st) | Set pendingBarStart | add_bar (pending) |
| add_bar (pending) | Click different node | Add bar between nodes, clear pending | add_bar |
| add_bar (pending) | Click same node | Clear pending (cancel) | add_bar |
| add_bar (pending) | Escape key | Clear pending | add_bar |
| select | Click node | Select node, show properties in side panel | select |
| select | Click bar | Select bar, show properties in side panel | select |
| select | Click empty canvas | Deselect all, close side panel | select |
| delete | Click node | Show confirmation dialog; on confirm: remove node + connected bars | delete |
| delete | Click bar | Show confirmation dialog; on confirm: remove bar | delete |
| any | Toolbar button click | Switch mode, clear selection and pending state | new mode |

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| 1 | Switch to Add Node mode |
| 2 | Switch to Add Bar mode |
| 3 | Switch to Select mode |
| 4 | Switch to Delete mode |
| Escape | Cancel pending bar / deselect / close side panel |
| Delete/Backspace | Delete selected element (with confirmation) |

### Animation Contract (from CONTEXT.md D-17)

| Animation | Duration | Easing | Stagger | Reduced Motion |
|-----------|----------|--------|---------|----------------|
| Diagram draw-in (stroke-dashoffset) | 600ms | ease-out | 80ms per bar | Instant (no animation) |
| Deformed shape draw-in | 800ms | ease-out | none (simultaneous) | Instant |
| Node appear (on add) | 150ms | ease-out | none | Instant |
| Side panel slide-in | 200ms | ease-out | none | Instant |

**Implementation:** CSS `stroke-dasharray` + `stroke-dashoffset` animation via `@keyframes drawDiagram`. Respect `prefers-reduced-motion: reduce` by setting `animation: none; stroke-dashoffset: 0`.
