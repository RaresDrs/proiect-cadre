  <action>
    **Pasul 1 — Creeaza directorul workflows:**
    ```bash
    mkdir -p C:/Users/Rares/proiect-cadre/.github/workflows
    ```

    **Pasul 2 — Creeaza `.github/workflows/frontend-ci.yml`:**
    ```yaml
    name: Frontend CI

    on:
      push:
        paths:
          - 'frontend/**'
          - '.github/workflows/frontend-ci.yml'
      pull_request:
        paths:
          - 'frontend/**'
          - '.github/workflows/frontend-ci.yml'

    jobs:
      build:
        name: Build React App
        runs-on: ubuntu-latest

        steps:
          - name: Checkout cod
            uses: actions/checkout@v4

          - name: Setup Node.js 24
            uses: actions/setup-node@v4
            with:
              node-version: '24'
              cache: 'npm'
              cache-dependency-path: frontend/package-lock.json

          - name: Instaleaza dependinte
            run: npm ci --prefix frontend

          - name: Build productie
            run: npm run build --prefix frontend

          - name: Verifica output build
            run: test -d frontend/dist && echo "Build OK — dist/ exista"
    ```

    **Pasul 3 — Creeaza `.github/workflows/backend-ci.yml`:**
    ```yaml
    name: Backend CI

    on:
      push:
        paths:
          - 'backend/**'
          - '.github/workflows/backend-ci.yml'
      pull_request:
        paths:
          - 'backend/**'
          - '.github/workflows/backend-ci.yml'

    jobs:
      test:
        name: Test FastAPI Backend
        runs-on: ubuntu-latest

        steps:
          - name: Checkout cod
            uses: actions/checkout@v4

          - name: Setup Python 3.12
            uses: actions/setup-python@v5
            with:
              python-version: '3.12'
              cache: 'pip'
              cache-dependency-path: backend/requirements.txt

          - name: Instaleaza dependinte
            run: pip install -r backend/requirements.txt

          - name: Ruleaza teste pytest
            run: pytest backend/tests/ -v
            env:
              PYTHONPATH: backend

          - name: Verifica import FastAPI app
            run: python -c "import sys; sys.path.insert(0,'backend'); from app.main import app; print('Import OK')"
    ```

    **Pasul 4 — Commit si push workflows pe GitHub:**
    ```bash
    cd C:/Users/Rares/proiect-cadre
    git add .github/workflows/frontend-ci.yml .github/workflows/backend-ci.yml
    git commit -m "ci: adauga GitHub Actions pentru frontend si backend monorepo"
    git push origin master
    ```

    **Pasul 5 — Verifica ca Actions au fost trigerate:**
    Dupa push, deschide https://github.com/[username]/proiect-cadre/actions si asteapta ca workflow-urile sa apara. NOTA: Daca repo-ul este privat, Actions functioneaza la fel.
  </action>
